# Image-Search-Engine
Image search engine in python.

**Technologies:**

Angular 5

Python 3

Postgresql 10


**Enable Cube**

CREATE EXTENSION cube;

**Install POSTGIS**

sudo apt-get install postgis
sudo apt-get install postgis-gui

CREATE EXTENSION gist;
